## Calendar-react

The site is available at https://youthful-hopper-99d314.netlify.app/

## Install

    $ git clone https://github.com/gon4aruk/calendar-react.git
    $ cd calendar-react
    $ npm install

## Start & watch

    $ npm start

## Build for production

    $ npm run build

**The tech stack is:**

- HTML5
- CSS3
- Flexbox
- Sass (Scss)
- BEM methodology
- JS(ES6)
- moment.js
- ReactJS
- React hooks
- Webpack
